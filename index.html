quicksilver-my by makotoy

QuickSilver 'gamma', a version of the widely influential & popular tool on Mac OS X.

QuickSilver is the widely influential & popular tool on Mac OS X. This project aims at maintaining the 'modern' QuickSilver codebase conceived by the original maintainer, but missed the release to the public, in an actually buildable & usable form. It was forked from the then 'trunk' tree at code.google.com/p/blacktree-alchemy .

This project has a wiki for developers' info.

Dependencies

no external dependancies

Install

Currently the code as it is requires Mac OS X 10.6 (Snow Leopard) and Xcode 3. With some minor changes it should be able to be built on 10.5 (Leopard). Please contact the maintainer for the details.
